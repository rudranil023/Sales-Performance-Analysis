# Sales Performance & Profitability Analysis

## Project Overview
This project analyzes sales data to identify revenue drivers, profitability trends, and city-level performance.

## Tools Used
- SQL
- Power BI
- Excel
- GitHub

## Key Insights
- Mumbai is the highest revenue and profit city.
- Electronics drives overall revenue.
- Furniture caused a revenue spike in Month 2.
- Kolkata shows margin optimization opportunity.

## Business Recommendations
- Introduce premium electronics category.
- Improve pricing strategy in Kolkata.
- Maintain volume growth without aggressive discounting.

## Dashboard Features
- Revenue Trend Analysis
- City Performance Comparison
- Category Contribution Analysis
- Profit & Margin Tracking
